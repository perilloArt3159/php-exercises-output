# Project Setup

There are two ways to run the project. Please note that all of these options require you to have Apache and PHP 8.1 installed.

* Option 1 : Put the files inside the *htdocs* root folder of your Xammp (windows) or Lamp (Linux or Mac) Installation then type `http:\\localhost` on the url bar.

* Option 2 : In the project root directory Run php's built in development server through the following command using a termintl then use the link provided.

```bash
# Start the Development Server
$ php -S localhost:3000

# The link would be http://localhost:3000
```
